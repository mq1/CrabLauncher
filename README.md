## Installation

### Package (recommended)

Just grab the [latest release](https://github.com/mq1/icy-launcher/releases/latest) for your platform

### From source

```sh
git clone https://github.com/mq1/icy-launcher.git
cd icy-launcher
cargo build --release
```
