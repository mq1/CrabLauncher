## Installation

### Package (recommended)

Just grab the [latest release](https://github.com/mq1/ice-launcher/releases/latest) for your platform

### From source

```sh
git clone https://github.com/mq1/ice-launcher.git
cd ice-launcher
cargo build --release
```
